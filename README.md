# pygit

Упрощённая реализация Git на Python, в которой с нуля воспроизводятся основные механизмы системы контроля версий.

## Описание
`pygit` — консольное приложение, которое эмулирует базовые команды Git, такие как `init`, `add`, `commit`, `log` и другие.  
Проект позволяет на практике разобраться, как Git работает на низком уровне: как инициализируется репозиторий, как отслеживаются файлы, сохраняется история изменений и организуется работа с ветками.

## Возможности
- инициализация репозитория
- добавление файлов в отслеживание
- создание коммитов
- просмотр истории коммитов
- базовая работа с ветками
- хранение файлов и служебных данных в собственной Git-подобной структуре

## Технологии
- Python
